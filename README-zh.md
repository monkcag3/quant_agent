## 策略上线流程
### 回测阶段
* 使用SimMd提供历史行情
* 使用SimTd提供交易服务

### 模拟阶段
* 使用HackerMd提供准实时行情
* 使用SimTd提供交易服务或者使用其他模拟盘工具

## 实盘阶段
* 使用HackerMd通过准实时行情
* 使用HackerTd进行实盘交易
